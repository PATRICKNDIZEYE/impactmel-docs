# Introduction

**ImpactMEL** is a full-stack enterprise Monitoring, Evaluation & Learning (M&E) platform built for development organizations, NGOs, and donors. It replaces fragmented spreadsheets and legacy tools like DevResults with a single, real-time system.

## What it does

| Capability | Description |
|---|---|
| **Results Framework** | Model Impact → Outcome → Output → Activity chains |
| **Indicator Management** | Define, target, and track indicators across projects and periods |
| **Data Collection** | Build and deploy custom forms; submissions feed indicators automatically |
| **Reporting** | Auto-assemble donor reports, program reviews, evidence packs |
| **Budget Tracking** | Track budget lines, burn rates, and financial summaries per project |
| **Multi-org / Multi-donor** | Separate orgs, each with programs spanning multiple funders |
| **RBAC** | Role-based access at org level down to individual report approval |

## Who uses it

- **M&E Officers** — define indicators, manage data collection cycles, review submissions
- **Program Managers** — track project progress, view dashboards, manage teams
- **Reporters / Field Staff** — submit data against assigned indicators for a period
- **Donors** — receive shareable public reports (no login required)
- **Org Admins** — configure the organization, manage members, set up settings

## Core philosophy

> One source of truth from field data entry to donor report.

Data flows in one direction:

```
Field Staff → Form Submission → Indicator Report → Report Assembly → Donor PDF
```

No copy-paste. No emailed spreadsheets. No reconciliation.
